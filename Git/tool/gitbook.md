# Gitbook

### 安装

准备好 node 然后全局安装 gitbook-cli\
`npm install gitbook-cli -g`

新建项目文件夹，进入文件夹在终端运行\
`gitbook init`

如果出现

```
info: create README.md
info: create SUMMARY.md
info: initialization is finished
```

就成功了

{% hint style="info" %}
如果有报错的话可能就是 node 版本问题 可以安装 nvm 去切换 node 版本或者直接安装最新/对应的版本，之后重新执行 `git init`
{% endhint %}
